### Development Environment Setup

---

**Clone the repository to your local machine**
:

```bash
git clone https://github.com/Sam44323/slippage_calculator.git
```

---

**Install the dependencies**
:

```bash
yarn install
```

---

**Run the dev-client**
:

```bash
yarn start
```

---

**Creating a production build**
:

```bash
yarn build
```
